Left sidebar layout for domain navigation

oh-my-mermaid has a 200px fixed left sidebar. Archon currently has no navigation. Which layout works best?

A

Fixed 200px sidebar (oh-my-mermaid style)

Pros: Familiar pattern, screen real estate predictable, matches your UX baseline.

Cons: Takes 200px from graph canvas on all screens.

B

Collapsible sidebar (default expanded)

Pros: Full navigation when needed, more canvas space when collapsed.

Cons: Adds UI complexity, hidden navigation is harder to discover.

C

Minimal overlay (no sidebar)

Pros: Maximum canvas space, simplest implementation.

Cons: No persistent navigation tree, harder to browse large graphs.
