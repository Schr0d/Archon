设计规格对比

NOC 监控室风格
  • 纯黑背景 (#000000) + 微弱网格
  • IBM Plex Mono (标题) + JetBrains Mono (数据)
  • 精确绿 #00FF41 = 健康
  • 警示橙 #FF6B00 = 风险/警告
  • 阻塞红 #FF0000 = 违规/阻塞
  • 高信息密度，像 Datadog 面板
  • 1px 精确线条，最小装饰
  • 状态驱动颜色，快速扫描
  • 脉冲动画表示活动节点
  • 适合长时间工作的暗色环境
技术蓝图风格
  • 浅灰蓝背景 (#f0f4f8 渐变)
  • IBM Plex Mono + 技术制图感
  • 工程蓝 #0066CC = 主色
  • 警告黄 #f59e0b = 注意
  • 清晰边框，2px 精确线条
  • 卡片式布局，阴影增加深度
  • 像架构师的工作台蓝图
  • 适合文档/报告展示
  • 打印友好
  • 更像"专业工具"而非"监控面板"

Which direction resonates more with how you work with dependency data?
